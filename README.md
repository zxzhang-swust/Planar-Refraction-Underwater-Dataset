This repository releases a dataset for planar refractive surface underwater imaging.
The real-world dataset contains 225 sub-aperture images captured by a Lytro Illum light field camera.
Given the small pitch of the microlenses inside the light field camera, we select 9 symmetric sub-aperture images with the relatively largest spacing.
The synthetic dataset is generated using the Taichi programming library, following the sub-aperture structure of the real dataset.
